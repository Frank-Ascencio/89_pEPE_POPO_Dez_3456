Download Polydot

  • Windows Instructions

    1. Download Voicemeeter Banana (this is recommended if you already have to virtual cables you can skip this)
    2. Once you download Voicemeeter in settings put "speaking output" as the MMO Voicemeeter Input and if you're using listening, for "listening input" put MMO Voicemeeter Aux Input
    3. After put Voicemeeter Output as the input in the voice chat application you're using and if you're listening put Voicemeeter Aux Output for the output device in the voice chat application

  • Mac Instructions

    1. Go to your Mac Terminal and type in "sudo spctl --master-disable", and when prompted, type in your password.
    2. Download Virtual audio cables (this is recommended if you already have to virtual cables you can skip this)
    3. Download Blackhole 2ch For a second pair of audio cables (do this for the listening function)
    4. After put virtual cable as the input in the voice chat application you're using and if you're listening put Blackhole2ch for the output device in the voice chat application
    5. Optional: if you still want to hear the user on zoom got to audio MIDI setup and then create a multi output device with your desire hardware output and with blackhole marked. After put this new mult output device as your output device on zoom
